Endangered Animals


  • This is a Anoa, or a midget buffalo. It's scientific name is a bubalus depressicornis.

  • This animal is located in Indonesia.

  • The anoa is endangered.


  • This is the Giant Stable Antelope. It's scientific name is Hippotragus niger variani.

  • This animal is located in the Southwest region of Africa.

  • The giant stable antelope is critically endangered animal.


  • This is a Desert Bandicoot. It's scientific name is parameles eremiana.

  • This animal is located in northeast South Australia and north central Western Australia.

  • The desert bandicoot is extinct.


  • This is a Mariana Fruit Bat. It's scientific name is pteropus mariannus.

  • This animal is located in the Marianas.

  • The mariana fruit bat is endangered.
